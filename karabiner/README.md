# Karabiner Elements

KE's config is located at `~/.config/karabiner/karabiner.json`.
Just copy the json file to the target.


```bash
# brew cask install karabiner-elements
mkdir -p ~/.config/karabiner
cp karabiner.json ~/.config/karabiner
```
